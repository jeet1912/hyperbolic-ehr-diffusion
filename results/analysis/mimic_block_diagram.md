Legend
------
- **Hyperbolic Code Embedding:** `HyperbolicCodeEmbedding` pretrained with $\lambda_{\text{radius}}$ and $\lambda_{\text{hdd}}$ losses.
- **Graph Diffusion Layer:** `HyperbolicGraphDiffusionLayer` using random-walk kernels derived from diffusion steps.
- **Global Self-Attention Stack:** `GlobalSelfAttentionBlock` modules inserted after mean pooling to merge global co-occurrence context before time embeddings.
- **TemporalLSTMEncoder:** Processes real visit latents, produces pooled states for classification and feature-consistency loss.
- **TrajectoryVelocityModel:** Implements rectified-flow loss, generates synthetic latents for $\lambda_S$ supervision.
- **RiskHead:** Shared linear head scoring both real and synthetic representations.

This block diagram mirrors architecture encoded in `risk_prediction_mimic.py`.
